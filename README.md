# Growth Tracker 🌱

> *追踪程序员的成长轨迹，让每一步进步都被记录。*

## 项目简介

Growth Tracker 是一个命令行工具，帮助开发者记录每日学习、设定阶段目标、追踪技能树进化。

## 功能特性

- 📅 **每日记录** - 快速记录今日所学
- 🎯 **阶段目标** - 设定并追踪长期目标
- 🌳 **技能树** - 管理技能掌握程度
- 📊 **数据统计** - 生成成长报告
- 🔗 **GitHub集成** - 关联开源贡献

## 安装

```bash
go install github.com/codex/growth-tracker@latest
```

## 快速开始

```bash
# 初始化配置
growth-tracker init

# 记录今日学习
growth-tracker log "学习了Go的并发模型"

# 查看成长报告
growth-tracker stats
```
